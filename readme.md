# Calculadora em Java

### Este projeto é uma implementação simples de uma calculadora com interface gráfica utilizando Java Swing. A calculadora permite realizar operações básicas como adição, subtração, multiplicação e divisão. 

# Estrutura do projeto 

### O projeto é composto por duas classes principais :

- `view :` Esta classe é responsável pela interface gráfica do usuário. Ela cria a janela da calculadora, o display para mostrar os números e resultados, e os botões para os números e operações. 

- `controller :` Esta classe lida com a lógica da calculadora. Ela gerencia os eventos dos botões e realiza as operações matemáticas conforme os botões são pressionados.
